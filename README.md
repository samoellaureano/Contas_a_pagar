# Contas a Pagar
## Desafio Backend
### Tecnologias Utilizadas
- Java 11
- Spring Boot
- Spring Data
- Spring Security
- Lombok
- Swagger
- Junit
- Docker

### Endpoints
- GET /api/v2/swagger-ui.html - Documentação da API

### Execução do Projeto Docker
```shell
docker build -t desafio-backend .
docker-compose up
```

### Execução do Projeto Local
```shell
mvn clean install
mvn spring-boot:run
```
