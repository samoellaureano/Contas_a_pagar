# Accounts Payable
## Backend Challenge
### Technologies Used
- Java 11
- Spring Boot
- Spring Data
- Spring Security
- Lombok
- Swagger
- Junit
- Docker

### Endpoints
- GET /api/v2/swagger-ui.html - Documentação da API

### Docker Project Execution
```shell
docker build -t desafio-backend .
docker-compose up
```

### Local Project Execution
```shell
mvn clean install
mvn spring-boot:run
```
